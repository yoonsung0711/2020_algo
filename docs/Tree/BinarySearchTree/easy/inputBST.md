## Two Sum IV - Input is a BST

Given a Binary Search Tree and a target number, return true if there exist two elements in the BST such that their sum is equal to the given target.

__Example 1:__

```ts
Input: 
    5
   / \
  3   6
 / \   \
2   4   7

Target = 9

Output: True
```
 

__Example 2:__

```ts
Input: 
    5
   / \
  3   6
 / \   \
2   4   7

Target = 28

Output: False
```
