##### 예제

```
1 -> 1  
    (1 - 1) / 3 
        = ..0  
        (=> ..0)
        => 1
2 -> 2  
    (2 - 1) / 3 
        = ..1   
        (=> ..1)
        => 2
3 -> 4     
    (3 - 1) / 3 
        = ..2   
        (=> ..2)
        = 4
4 -> 11      
    (4 - 1) / 3 
        = 1..0 
        (=> 0..0)
        => 11
5 -> 12      
    (5 - 1) / 3 
        = 1..1 
        (=> 0..1)
        => 12
6 -> 14  
    (6 - 1) / 3 
        = 1..2
        (=> 0..2)
        => 14
7 -> 21  
    (7 - 1) / 3 
        = 2..0
        (= 1..0)
        = 21
8 -> 22  
    (8 - 1) / 3 
        = 2..1
        (=> 1..1)
        => 22
9 -> 24  
    (9 - 1) / 3 
        = 2..2
        (=> 1..2)
        => 24
10 -> 41  
    (10 - 1) / 3 
        = 3..0
        (=> 2..0)
        => 41
11 -> 42  
    (11 - 1) / 3 
        = 3..1
        (=> 2..1)
        => 42
12 -> 44  
    (12 - 1) / 3 
        = 3..2
        (=> 2..2)
        => 44
13 -> 111  
    (13 - 1) / 3 
        = 4..0
        (=> 3..0)
        = 1..0..0
        (=> 0..0..0)
        => 111
14 -> 112  
    (14 - 1) / 3 
        = 4..1
        (=> 3..1)
        = 1..0..1
        (= 0..0..1)
        => 112
15 -> 114
    (15 - 1) / 3 
        = 4..2
        (=> 3..2)
        = 1..0..2
        (=> 0..0..2)
        => 114
16 -> 121
    (16 - 1) / 3 
        = 5..0
        (=> 4..0)
        = 1..1..0
        (=> 0..1..0)
        = 121
17 -> 122  
    (17 - 1) / 3 
        = 5..1
        (=> 4..1)
        = 1..1..1
        (=> 0..1..1)
        => 122

    (18 - 1) / 3 
        = 5..2
        (=> 4..2)
```
